About

This webservice was developed by Dennis Demmer with the Java Webframework Spring Boot.
For the named entity recognition the Python API SpaCy (version 2.0.18) was used.
It's running on a Raspberry Pi 3 B+ with Raspian Stretch Lite OS installed.
You can download and use the source code for this web application from github.com/DDemmer1/NamedEntityRecognitionWebservice

REST API

This web application also includes a REST API.
You can use it for free. But since the service is running on a Raspberry Pi don't expect any spectacular performance.

Example:
http://dennis-demmer.de/api?text=This+is+a+test+by+Dennis+Demmer+from+Cologne%2C+Germany.&format=json&language=en

Acceptable parameter inputs are:
- language = de, fr, en
- format = json, xml
